# IPProxys
IPProxys代理池项目，提供代理ip
<br/>
详细使用方式，请看我的博客:
http://www.cnblogs.com/qiyeboy/p/5693128.html
<br/>
linux debian下:
<br/>
需要安装sqlite数据库:
apt-get install sqlite sqlite3
<br/>
安装requests库
pip install requests
<br/>
安装lxml
apt-get install python-lxml
